5. Conclusiones y Referencias

Conclusiones

La filosofía aplicada en el manejo de base de datos relacionales y NoSQL es muy diferente sin embargo ambas tienen utilidad dependiendo del universo que se quiera aplicar.

La flexibilidad que ofrecen las bases de datos no relacionales es el mayor beneficio para un universo cambiante dentro de un negocio.

Crear usuarios en MongoDB requiere la activación del modo de autenticación.

Crear usuarios con roles predefinidos es simple usando el método db.createUser.
